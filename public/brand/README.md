# Brand assets

## afton-logo.png  ← REQUIRED

Save the official Afton Fitness Equipment logo here as **`afton-logo.png`**
(transparent background PNG, or `.svg` — if you use SVG, update `LOGO_SRC`
in `src/components/layout/Logo.tsx`).

The header/footer pick it up automatically. Until the file exists, the site
falls back to a plain text wordmark so nothing appears broken.

Do not recolour or redraw the artwork. On dark surfaces the logo is placed in
a white padded container rather than inverted.
